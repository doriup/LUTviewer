# LUTviewer

LUTviewer is a macOS application designed for viewing and exporting LUTs (Look-Up Tables).

## Features

- **LUT Visualization**: Load and view 3D LUTs.
- **Image Support**: Support for various image formats, including DPX.
- **LUT Export**: Export LUTs for use in other applications.
- **Reference Library**: Manage and compare reference images.

## Development

The project is built using Swift and SwiftUI for macOS.

### Requirements

- macOS 13.0+
- Xcode 15.0+
