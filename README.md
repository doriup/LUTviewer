# LUTviewer

![LUTviewer Screenshot](screenshot.png)

LUTviewer is a macOS application designed for viewing and exporting LUTs (Look-Up Tables).

Special thanks to **Brice Barbier** for the idea and inspiration for this project!

## Features

- **LUT Visualization**: Load and view 3D LUTs.
- **Image Support**: Support for various image formats, including DPX, TIFF, etc.
- **LUT Export**: Export LUTs for use in other applications.
- **Reference Library**: Manage and compare reference images.

## Installation & Security

Since this application is not signed with an Apple Developer account, macOS may block it from opening by default. To open it:

1.  **Right-click** (or Control-click) the application icon and select **Open**.
2.  Click **Open** in the dialog that appears.
3.  Alternatively, if it still won't open, go to **System Settings > Privacy & Security** and click **Open Anyway** at the bottom.

## Download
